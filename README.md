## Project Overview
Customer churn is a critical metric for financial institutions. This project aims to predict whether a bank customer will close their account using a machine learning classification model. By accurately identifying at-risk customers, businesses can proactively deploy retention strategies, minimizing opportunity loss and improving customer lifetime value.

## Tech Stack
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Machine Learning:** Scikit-Learn, XGBoost, Imbalanced-Learn (SMOTE)
* **Data Visualization:** Matplotlib, Seaborn


## Business Impact
By deploying this predictive model, a business can segment customers based on their churn probability. This allows marketing and operations teams to focus retention budgets specifically on high-risk, high-value customers, directly impacting the bottom line.
